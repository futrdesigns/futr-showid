# Simple Show-ID Script for QB Core 

A clean and simple FiveM script that displays player IDs for QB Core/QBox servers.

##  Features

- **Simple ID Display**: Shows only player IDs (no extra information)
- **QB Core Compatible**: Designed specifically for QB Core/QBox framework
- **Multiple Access Methods**:
  - Target system (qb-target)
  - Chat command (/myid)
  - Optional keybind (F5)
- **Visual Feedback**: Optional animations and notifications
- **Cooldown System**: Prevents spam
- **Floating Text**: Optional 3D text above players' heads
